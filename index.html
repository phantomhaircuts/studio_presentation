Types of Attacks

Q. What is SQL injection?
A. A SQL injection attack makes use of vulnerable user inputs in an application, and sends a malicious payload via SQL query. Attacks can potentially get user credentials, drop or delete a database, alter or add data etc. Q. What is a DDoS Attack?
A. A DoS or Denial of Service attack, is when an attacker renders a service unusuable and inaccessible by indefinitely disrupting services of a host. This attack is accomplished by flooding a target machine or resource with requests until it overloads the systems. A Distributed denial of service attack, distributes the incoming flood of traffice from multiple sources making it impossible to prevent the attack from simply blocking a single source. Q. What is HTTP hacking?
A. Cross Site forgery attacks happen when a hacker sends a forged HTTP request to collect cookie information. Once the user’s session is compromised the hacker can generate requests to application identically to the user.
